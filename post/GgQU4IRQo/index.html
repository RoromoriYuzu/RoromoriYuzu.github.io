安装环境

对于中国大陆用户，建议将 NPM 源设置为国内的镜像，可以大幅提升安装速度。

$ npm install -g cnpm --registry=https://registry.npmmirror.com

$ cnpm install [name]

安装npm

去Node.js的官网下载安装程序

不必勾选，一路Next

安装Vue

# 最新稳定版
$ npm install vue@next

安装命令行工具 (CLI)

CLI 工具假定用户对 Node.js 和相关构建工具有一定程度的了解。如果你是新手，我们强烈建议先在不用构建工具的情况下通读指南，在熟悉 Vue 本身之后再使用 CLI。

$ npm install -g @vue/cli

之后执行命令查看是否安装成功

$ vue -V

@vue/cli 5.0.8

创建项目

  1. 新建一个存放项目文件的目录，在其中执行
$ vue create <project-name>

  1. 或者你可以执行以下命令，启动一个GUI窗口进行项目管理
$ vue ui

启动项目

 $ cd <project-name>
 $ npm run serve

之后就可以看到Vue大大的Logo了

使用Element UI

基于 Vue 2.x 的 Element UI

  1. npm 安装
$ npm i element-ui -S

  1. 引入 Element

    完整引入，在 main.js 中写入以下内容：

    import Vue from 'vue';
import ElementUI from 'element-ui';
import 'element-ui/lib/theme-chalk/index.css';
import App from './App.vue';

Vue.use(ElementUI);

new Vue({
  el: '#app',
  render: h => h(App)
});

    以上代码便完成了 Element 的引入。需要注意的是，样式文件需要单独引入。

Vue 3.x 的升级版本

  1. npm 安装
   $ npm install element-plus --save

  1. 引入 Element

    如果你对打包后的文件大小不是很在乎，那么使用完整导入会更方便。

    // main.ts
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')
